My jQuery basic assignment!



.CLICK

Vincula un controlador de eventos al evento de Javascript "click" o activa ese evento en un elemento

.Hide

Oculta los elementos coincidentes

.Show

Muestra los elementos coincidentes

.Toggle

Muestra u Oculta los elementos coincidentes

.Slide Dow

Muestra los elementos emparejados con un movimiento deslizante

.Slide Up

Oculta elementos emparejados con un movimiento

.Slide Toggle

Muestra u oculta los elementos emparejados con un movimiento

.Fadeln

Muestra los elementos coincidentes atenuandolos a opaco

.Fadout

Oculta los elementos coincidentes atenuandolos a transparentes

.AddClass

Agrega la clase() espcificada para cada elemento en el conjunto de elementos coincidentes

.Before

Inserte contenido, especificado por el parámetro, antes de cada elemento en el conjunto de elementos coincidentes.

.After

Insertar contenido, especificado por el parámetro, después de cada elemento en el conjunto de elementos coincidentes.

.Append

Inserte contenido, especificado por el parámetro, al final de cada elemento en el conjunto de elementos coincidentes

.HTML

Obtenga el contenido HTML del primer elemento del conjunto de elementos coincidentes o configure el contenido HTML de cada elemento coincidente.

.Attr

Obtenga el valor de un atributo para el primer elemento en el conjunto de elementos coincidentes o establezca uno o más atributos para cada elemento coincidente.

.Val

Obtenga el valor actual del primer elemento en el conjunto de elementos coincidentes o establezca el valor de cada elemento coincidente.

.Text

Obtenga el contenido de texto combinado de cada elemento en el conjunto de elementos coincidentes, incluidos sus descendientes, o establezca el contenido de texto de los elementos coincidentes.
